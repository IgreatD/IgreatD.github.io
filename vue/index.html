# Vue-Cli 脚手架

  • 使用npm安装
npm install -g @vue/cli
  • 创建一个项目
vue create [project_name]
  • cd到项目目录 安装依赖
npm install
  • 项目运行 (开发环境)
npm run serve
  • 项目编译 (生成环境)
npm run build
