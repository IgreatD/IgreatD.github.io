# mkdir 命令

常见目录命令。

  • 命令名称：mkdir。
  • 英文原意：make directories。
  • 所在路径：/bin/mkdir。
  • 执行权限：所有用户。
  • 功能描述：创建空目录。

  • 命令格式
  • 常见用法
    • 1. 建立一个目录
    • 2. 递归创建多个目录

# 命令格式

[root@localhost ~]# mkdir [选项][目录名]

选项：

  1. -p 递归创建

# 常见用法

# 1. 建立一个目录

mkdir cangls

# 2. 递归创建多个目录

mkdir -p a/b/c/d
