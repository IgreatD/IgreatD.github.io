# ls 命令

目录操作命令，主要作用是显示目录下的内容。

  • 命令名称：ls

  • 英文名：list

  • 所在目录：/bin/ls

  • 执行权限：所有用户

  • 功能描述：显示目录下的内容

  • 命令格式
  • 常见用法
    • 1. -a
    • 2. -l
    • 3. -d
    • 4. -h
    • 5. -i

# 命令格式

[root@localhost ~]# ls [选项][文件名或者目录名]

选项：

  • -a：显示所有文件

  • -l：长格式显示

  • -d：显示目录信息，而不是目录下的文件

  • -h：人性化显示，显示文件大小

  • -i：显示文件的i节点号

# 常见用法

# 1. -a

可以显示隐藏的文件

ls -a
# .  ..  bin  boot  dev  ...

# 2. -l

显示文件的详细信息：

  1. 权限

  2. 引用计数。文件的引用计数代表该文件的硬链接个数，而 目录的引用计数代表该目录有多少个子目录

  3. 所有者。这个文件属于哪个用户，默认为创建这个文件的用户。

  4. 所属组。

  5. 大小。默认单位是字节

  6. 文件修改时间。

  7. 文件名

ls -l
# total 0
# drwxr-xr-x. 8 root    root    158 Mar 15 21:24 express
# drwx------. 2 igreatd igreatd  62 Apr 11  2018 igreatd
# drwxr-xr-x. 2 root    root     22 Mar 15 22:26 src

# 3. -d

显示目录的详细信息

ls -ld /home/
# drwxr-xr-x. 5 root root 67 Mar 15 22:26 /home/

# 4. -h

显示符合我们习惯的文件大小

ls -lh
# total 36K
# -rw-r--r--.  1 root root 1.1K Mar 15 02:13 app.js
# drwxr-xr-x.  2 root root   17 Mar 15 02:32 bin
# -rw-r--r--.  1 root root 2.6K Mar 15 03:37 nginx.conf
# drwxr-xr-x. 94 root root 4.0K Mar 15 02:34 node_modules
# -rw-r--r--.  1 root root  299 Mar 15 02:13 package.json
# drwxr-xr-x.  5 root root   58 Mar 15 02:32 public
# drwxr-xr-x.  2 root root   38 Mar 15 02:32 routes
# drwxr-xr-x.  2 root root   37 Mar 15 21:47 tcp
# drwxr-xr-x.  2 root root   61 Mar 15 02:32 views
# -rw-r--r--.  1 root root  20K Mar 15 02:17 yarn.lock

# 5. -i

每个文件都有一个被称作 inode（i 节点）的隐藏属性，可以看成系统搜索这个文件的 ID，而-i选项就是用来査看文件的 inode 号的
