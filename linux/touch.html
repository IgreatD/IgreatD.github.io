# touch 命令

如果文件不存在，则会创建空文件。如果存在，则会修改文件的时间戳

  • 命令名称：touch。
  • 英文原意：change file timestamps。
  • 所在路径：/bin/touch。
  • 执行权限：所有用户。
  • 功能描述：修改文件的时间戳。

注意

千万不要把它作为创建文件的命令

  • 命令格式

# 命令格式

[root@localhost ~]# touch [选项]文件或者文件夹

选项：

  • -a：只修改文件的访问时间

  • -c：如果文件不存在，则不建立新文件

  • -d：把文件的时间改为指定的时间

  • -m：只修改文件数据的修改时间

在Linux中，每个文件都有三个时间，分别是访问时间(Access Time)、数据修改时间(Modify Time)和状态修改时间(Change Time)

不过，touch 命令只能手工指定只修改访问时间，或是只修改数据修改时间，而不能指定只修改状态修改时间。因为不论是修改访问时间，还是修改文件的数据时间，对文件来讲，状态都会发生改变，即状态修改时间会随之改变。

ls -ld express
# drwxr-xr-x. 8 root root 158 Mar 15 21:24 express
touch express
ls -ld express
# drwxr-xr-x. 8 root root 158 Mar 16 02:04 express

注意

在Linux中，没有文件创建时间
