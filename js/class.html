# contructor 方法

  • contructor方法是类的默认方法，通过new命令生成新的对象时，自动调用该方法。

  • 一个类必须有contructor方法，如果没有显示定义，一个空的contructor会被默认调用。

  • contructor默认方法类的实例对象this。

class Foo{
    constructor(){
        console.log(this) // Foo {}
    }
}

let foo = new Foo()

console.log(foo instanceof Foo) // true

注意

类必须使用new调用

# 类的实例

  • 通过new方法生成类的实例对象

  • 类的实例属性除非显式定义在本身this上，否则都定义在原型上prototype

class Foo {

    constructor(x, y) {
        this.x = x
        this.y = y
    }

    toString() {
        console.log('x =', this.x, ', y =', this.y)
    }

}

let foo = new Foo(1,2)

foo.toString() // x = 1 , y = 2

console.log(foo.hasOwnProperty('x')) // true
console.log(foo.hasOwnProperty('y')) // true
console.log(foo.hasOwnProperty('toString')) // false
console.log(foo.__proto__.hasOwnProperty('toString')) // true
console.log(Foo.hasOwnProperty('toString')) // false
console.log(Foo.prototype.hasOwnProperty('toString')) // true

# 取值函数(getter)和存值函数(setter)

在类的内部可以使用get和set关键字，对某个属性设置取值函数和存值函数，拦截该属性的存取方法

# 属性表达式

类的属性名可以使用表达式来定义

let methodName = 'method_name'

let symbolMethodName = Symbol('symbol_method_name')

class Foo {

    [methodName]() {
        console.log('method_name') // method_name
    }

    [symbolMethodName](){
        console.log('symbol_method_name') // symbol_method_name
    }
}

let foo = new Foo()

foo[methodName]()

foo[symbolMethodName]()
