# Timer 定时器

timer 模块暴露了一个全局的 API，用于预定在将来的某个时间段调用的函数。因为定时器是全局函数，所以不需要require来使用。

  • Immediate
    • setImmediate(callback[...args])
  • Timeout
    • setTimeout(callback, delay[, ...args])
    • setInterval(callback, delay[, ...args])

# Immediate

此对象从内部创建，并从setImmediate返回。它可以传给clearImmediate来取消此操作。

# setImmediate(callback[...args])

  • callback：在当前回合的事件循环完成之后的回调函数

  • ...args：传给callback回调函数的参数，可选

  • 返回一个immediate实例对象，可以传递给clearImmediate来取消此函数

预定在 I/O 事件的回调之后立即执行的 callback。

const util = require('util');
const setImmediatePromise = util.promisify(setImmediate);

setImmediatePromise('foobar').then((value) => {
    console.log('在 I/O 回调之后', value)
});

function timerExample() {
    console.log('在 I/O 回调之前');
    setImmediatePromise();
    console.log('在 I/O 回调之后');
}
timerExample();

/*
在 I/O 回调之前
在 I/O 回调之后
在 I/O 回调之后 foobar
*/

ref 和 unref

默认情况下，所有 Immediate 对象都是 ref 的，通常不需要调用 immediate.ref()，除非之前调用了 immediate.unref()

# Timeout

此对象在内部创建，并从setTimeout()和setInterval()返回。 它可以传给clearTimeout()或clearInterval()以取消计划的操作。

# setTimeout(callback, delay[, ...args])

  • callback 当定时器到时间时执行的函数

  • delay 调用callback延迟的时间

  • ...args 调用callback可选的参数

  • 返回timeout的实例对象

预定在 delay 毫秒之后执行一次性的 callback。

setTimeout((value) => {
    console.log('the callback will trigger after 2seconds')
    console.log('the callback argumnets are:', value)
}, 2000, 'footer');
// the callback will trigger after 2seconds
// the callback argumnets are: footer

或者使用util.promisify方法

const util = require('util')

const timeoutPromise = util.promisify(setTimeout)

timeoutPromise(2000, 'footer').then((value) => {
    console.log('the callback will trigger after 2seconds')
    console.log('the callback argumnets are:', value)
})
// the callback will trigger after 2seconds
// the callback argumnets are: footer

ref 和 unref

默认情况下，当使用 setTimeout() 或 setInterval() 预定定时器时，只要定时器处于活动状态，Node.js 事件循环将继续运行。 这些函数返回的每个 Timeout 对象都会导出 timeout.ref() 和 timeout.unref() 函数，这些函数可用于控制此默认行为。

# setInterval(callback, delay[, ...args])

  • callback 当定时器到时间时执行的函数

  • delay 调用callback延迟的时间

  • ...args 调用callback可选的参数

  • 返回timeout的实例对象

预定每隔 delay 毫秒重复执行 callback。

当 delay 大于 2147483647 或小于 1 时， delay 将设置为 1。

let count = 1
let intervalInstance = setInterval(() => {
    console.log('this callback will loop 1seconds', count)
    count++
    if (count > 10) {
        clearInterval(intervalInstance)
    }
}, 1000);

如果 callback 不是函数，则抛出 TypeError。

setInterval('footer', 1000);
// TypeError [ERR_INVALID_CALLBACK]: Callback must be a function
